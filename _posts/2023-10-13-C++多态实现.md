## 虚函数、虚函数表 和 虚函数表指针
假设有基类 Device 和子类 Phone：
```
class Device
{
};

class Phone
{
};
```

编译器会在编译阶段，为含有虚函数的类自动加上一个void*成员vptr，即虚函数表指针，它会指向虚函数表。每个具有虚函数的类就会产生一个虚函数表，在编译器生成。比如上面这种情况，Device和Phone分别会有一个虚函数表。虚函数表，可以看成一个数组vtbl，里面的每一项是一个函数地址。所以如果调用的是虚函数，那么运行时是怎么去虚函数表定位到正确的函数地址的呢？答案是每个虚函数会根据它们声明的顺序，有个偏移量，这个偏移量就对应于它们在虚函数表里的下标位置，所以，当调用虚函数时，就是会根据偏移量去查找虚函数表，进行调用，完成！
